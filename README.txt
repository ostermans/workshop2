# Прогнозирование покупок для интернет-магазина

## Описание проекта
В этом проекте создана модель для предсказания покупок клиентов интернет-магазина в ближайшие 90 дней. Целью является улучшение работы маркетинга интернет-магазина и увеличение продаж.

## Данные
Использованы датасеты, включающие информацию о покупках и сообщениях от магазина. Целевой признак — совершит ли клиент покупку в течение 90 дней (классификация 0/1).

## Основные этапы
1. Предобработка данных: удаление выбросов, проверка дубликатов.
2. Создание признаков: количество покупок, средний чек, дни с последней покупки, любимые категории и т.д.
3. Перебор гиперпараметров с помощью **GridSearch**. Наилучшей моделью оказалась **CatBoostClassifier**
4. С использованием лучшей модели, на тестовой выборке получили метрику ROC-AUC 0.72.

## Результаты
Наиболее значимыми признаками оказались:
- Количество покупок
- Средний период между покупками
- Средняя цена
- Дни с последней покупки

Модель может быть использована для прогнозирования, кому из клиентов стоит отправить персонализированные предложения.
